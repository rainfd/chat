## chatroom
---

一个利用Django中的channels完成的websocket聊天室。在channels教程范例的基础上，添加了用户进出信息和实时用户列表。

### 安装&运行

```python
sudo apt-get install redis-server
pip install django
pip install channels
pip install asgi-redis
python manage.py runserver
```
